# decks/

Slide-deck source. Decks are HTML (reveal.js or equivalent) that fetch their content from `../operations/` and `../datasets/` the same way cockpit views do (HR-MSc-9).

See `../charter/REPO_STRUCTURE.md` and `../charter/HARD_RULES.md`.

## Contents

This directory is empty at Batch 1. The Batch 8 commit scaffolds:

```text
decks/
├── curriculum/<module-slug>/slides.html
├── viva/<capability-slug>/slides.html
├── transfer/<transfer-slug>/slides.html
└── shared/
    ├── reveal-theme.css
    └── deck-data-fetch.js
```

`.pptx` exports are generated by `../.github/workflows/decks-export.yml` and offered as workflow artifacts, not committed.
