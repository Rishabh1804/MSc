# cockpit/

PWA source for the CodeMike operating cockpit. Each cockpit view is an HTML shell that fetches data from `../operations/*.json` or `../datasets/*.csv` at runtime. No academic claim is hardcoded in cockpit HTML (HR-MSc-9).

See `../charter/REPO_STRUCTURE.md` and `../charter/HARD_RULES.md` for the rules that bind this directory.

## Contents

This directory is empty at Batch 1. The Batch 7 commit scaffolds the PWA shell, design tokens, and the first cockpit views. Build output ships to `../docs/` for GitHub Pages.
