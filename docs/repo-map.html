Working decision

The repository should not be reorganized by random moves. The safe cleanup path is:

  1. Use this map as the visual front door.
  2. Use REPO_STRUCTURE.md for placement rules.
  3. Batch future changes by phase.
  4. Migrate or archive messy files only in planned cleanup branches.
