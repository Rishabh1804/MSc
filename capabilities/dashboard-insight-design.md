# Capability — Dashboard Insight Design

## What this capability does

Converts data, metrics, and analysis into decision-support dashboard structures with clear KPIs, audience context, visual choices, interpretation, and action guidance.

## Inputs

- Decision context
- Target user
- KPI definitions
- Dataset or data source
- Refresh needs
- Visualisation constraints
- Device or PWA context

## Outputs

- KPI definitions
- Dashboard structure
- Visualisation plan
- Interpretation layer
- Data refresh notes
- QA checklist

## Method

Use `patterns/dashboard-kpi-pattern.md`.

Core sections:

1. Decision context
2. Audience
3. KPI definition
4. KPI quality check
5. Visual selection
6. Layout
7. Interpretation layer
8. Data and refresh
9. Mobile/PWA considerations
10. QA checklist
