# DES-001 Syllabus — Design Foundations

## Course description

DES-001 introduces UI/UX design foundations and applies them to CodeMike's Destination Master Browser.

The course focuses on practical design literacy: reading sources, comparing frameworks, extracting principles, executing design audits, and producing repository artifacts.

## Prerequisites

- Basic HTML/CSS/JavaScript awareness
- Familiarity with the Destination Master Browser artifact
- Willingness to compare multiple sources rather than rely on one article

## Course outcomes

By the end of DES-001, CodeMike should be able to:

1. distinguish UI from UX
2. explain core design principles
3. compare multiple design sources
4. translate design theory into interface decisions
5. audit a data-review dashboard
6. document evidence for grading and revision

## Primary artifact

```text
docs/design-foundations-v2.html
```

## Assessment components

| Component | Weight |
|---|---:|
| Deep reading notes | 30% |
| Source comparison | 20% |
| Practical lab execution | 25% |
| Dashboard/checklist integration | 15% |
